A relaxing web experience combining mesmerizing fractal animations with ambient meditation music.

## Features

- **Beautiful Fractal Animations**: Watch the infinite complexity of Mandelbrot and Julia sets unfold
- **500x500 Canvas**: Smooth, color-cycling fractals with zoom animation
- **Ambient Meditation Music**: Web Audio API synthesizes calming drone pads and harmonics
- **Two Fractal Types**: 
  - Mandelbrot Set with infinite zoom
  - Julia Set with dancing patterns
- **Zero Dependencies**: Pure HTML, CSS, and JavaScript

## Usage

1. Open `index.html` in a modern web browser
2. Click "Start Experience" to begin the meditation
3. Watch the fractals evolve while listening to ambient sounds
4. Use "Switch Fractal" to alternate between Mandelbrot and Julia sets
5. Click "Stop" when you're done

## How It Works

### Fractals
- **Mandelbrot Set**: Classic fractal with smooth color cycling and gradual zoom
- **Julia Set**: Dynamic fractal patterns with ethereal movement
- Both use smooth coloring algorithms for beautiful gradients

### Audio
- Synthesized ambient music using Web Audio API
- Layered sine wave oscillators create drone pads
- Low-frequency oscillators (LFOs) add subtle movement
- Harmonic series (1st, 2nd, 3rd, 5th harmonics) for rich texture
- High-frequency shimmer for ethereal quality

## Tech Stack

- HTML5 Canvas for rendering
- Web Audio API for sound synthesis
- Pure JavaScript (no frameworks)
- CSS3 for styling

## Browser Compatibility

Works in all modern browsers that support:
- HTML5 Canvas
- Web Audio API
- ES6 JavaScript
